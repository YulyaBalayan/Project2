# Учебный сайт гостиницы "Уют"

Простой учебный сайт отеля, созданный для изучения HTML и JavaScript начинающими разработчиками.

## Структура проекта

### Основные страницы
- **index.html** - Главная страница с общей информацией об отеле
- **rooms.html** - Страница с описанием номеров
- **services.html** - Страница с услугами отеля
- **gallery.html** - Галерея фотографий
- **contacts.html** - Контактная информация и формы

### JavaScript страницы
- **dom.html** - Демонстрация работы с DOM
- **latin.html** - Интерактивные элементы с addEventListener
- **task.html** - Решение алгоритмической задачи

## Особенности проекта

### Навигация
- Единое навигационное меню на всех страницах
- Внутренняя навигация с якорными ссылками
- Ссылки на внешние ресурсы (открываются в новом окне)
- Выделение активной страницы

### HTML элементы
- **Таблицы** с фоном, изображениями и ссылками в ячейках
- **Списки** (нумерованные, маркированные, вложенные)
- **Изображения** с миниатюрами и ссылками на полный размер
- **Формы** с различными типами полей

### Формы
1. **Форма обратной связи** (contacts.html):
   - Имя, фамилия, группа, телефон, email, дата
   - Метод POST, action="https://example.com/submit"

2. **Форма бронирования** (contacts.html):
   - Поля: text, number, radio, checkbox, date, password, hidden
   - Метод POST, action="https://example.com/submit"

### JavaScript функциональность

#### dom.html
- Работа с `document.getElementsByTagName()`
- Использование `.innerText`
- Демонстрация методов DOM
- Интерактивные примеры

#### latin.html
- Использование `addEventListener('click', ...)`
- Изменение данных на латинские символы по клику
- Логирование событий
- Различные типы событий (click, mouseover, mouseout, dblclick)

#### task.html
- Решение задачи "Номер места в плацкартном вагоне"
- Алгоритм определения верхнее/нижнее и купе/боковое место
- Интерактивная форма с `<input type="number">`
- Визуализация результата через JavaScript

## Технические требования

- **HTML5** - семантическая разметка
- **CSS3** - встроенные стили в теге `<style>`
- **JavaScript** - без фреймворков, чистый JS
- **Относительные пути** - без серверных зависимостей
- **Комментарии** - объяснение ключевых элементов

## Алгоритмическая задача

**Задача 1005: Номер места в плацкартном вагоне**

Определить по номеру места (1-54):
- Верхнее или нижнее место (нечетные/четные номера)
- Купе или боковое место (1-36 купе, 37-54 боковые)

### Примеры:
- Место 1: Верхнее, Купе
- Место 2: Нижнее, Купе  
- Место 37: Верхнее, Боковое
- Место 54: Нижнее, Боковое

## Использование

1. Откройте `index.html` в браузере
2. Используйте навигационное меню для перехода между страницами
3. На JavaScript страницах взаимодействуйте с элементами
4. Заполняйте формы для демонстрации их работы

## Образовательная ценность

Проект демонстрирует:
- Основы HTML разметки
- Работу с CSS стилями
- JavaScript DOM манипуляции
- Обработку событий
- Алгоритмическое мышление
- Структурирование веб-проекта
